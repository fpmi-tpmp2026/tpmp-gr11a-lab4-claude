# Project Name: Autopark Management System

## Description
Система управления грузовым автопарком, разработанная на языке C с использованием базы данных SQLite3. Приложение позволяет вести учет водителей, транспортных средств и заказов, а также автоматически рассчитывать заработную плату. Система включает в себя строгую проверку бизнес-логики, например, контроль перегруза машин.

## Installation
1. Установите зависимости (gcc, cmake, sqlite3-dev):
   `sudo apt install build-essential cmake libsqlite3-dev`
2. Склонируйте репозиторий:
   `git clone https://github.com/Nastya404/tpmp-lab4`
3. Соберите проект
   `make`
4. Перейдите в папку проекта и создайте директорию сборки:
   `mkdir build_test && cd build_test`
5. Скомпилируйте тесты для проекта:
   `cmake .. && make`

## Usage
Запустите исполняемый файл:
`./bin/autopark`
Используйте консольное меню для навигации:
- Пункты 1-3: Добавление данных.
- Пункты 6-12: Генерация отчетов и статистики.
